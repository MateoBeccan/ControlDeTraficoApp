Sistema de Gestión de Tránsito
Proyecto desarrollado a partir de un ejercicio académico de normalización de base de datos de la materia Bases de Datos. A partir de ese trabajo inicial se construyó un sistema completo que permite gestionar vehículos, personas, accidentes y multas.

El proyecto está desarrollado con:

Backend: Spring Boot (Java)

Base de datos: MySQL

Frontend: Vue.js

El sistema incluye:

Registro y gestión de vehículos y personas

Control de accidentes y multas

Panel administrativo intuitivo

Login exclusivo para administradores

Es un proyecto orientado al aprendizaje práctico, integrando conceptos de backend, bases de datos y desarrollo web.
